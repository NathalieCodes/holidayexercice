Learn HTML

Intro

HTML stands for Hyper Text Markup Language. It's a standard markup language used to create web pages and web applications (along with CSS). Today, it's built using semantic tags mainly that describe the structure of a page. Browsers do not display the HTML tags, but use them to interpret the content of the page.

Example 

 <!DOCTYPE html>
 <html lang="en" xmlns="http://www.w3.org/1999/html">
 <head>
     <meta charset="UTF-8">
     <title>HTML</title>
     <link rel="stylesheet" type="text/css" href="main.css">
 </head>
 <body>
    <div>
        Hey this is an example
    </div>
 </body>
 </html>

Tags

Inline versus Block elements:
All HTML elements have a default display depending on what type of element it is. A block element always starts on a new line and takes up the full width available (e.g: div) An inline element does not start on a new line and only takes up as much width as necessary (e.g: span).

Div tags versus Section/Aside/Header/Footer:
The importance of having semantic tags is that is allows search engines to index the content of your website, hence having better SEO results.

Images:
There are 3 types of formats - jpg, png and gif. You will use jpg when displaying pictures. Png is used for illustrations/graphs contents because it allows transparency. Gif are used for animated pictures.
