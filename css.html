Learn CSS

Intro

CSS stands for Cascading Style Sheets. It's a language that describes the style of an HTML page. It describes how the different elements should be displayed in the page (size, color, location in the page...). It uses Selectors and Classes to specify which HTML element is targeted.

Example 

nav {
    display: flex;
    justify-content: space-between;
    flex-direction: column;
    align-items: center;
}

.title a {
    display: inline-block;
    letter-spacing: -2px;
    color: aqua;
    font-weight: bolder;
    font-size: 2rem;
    padding: 10px 20px;
    text-decoration: none;
}

Cascading

The term cascading means that when you have several displaying information for the same element, the last CSS information will be applied. Now, this is not always true! There is also a point system. The more Selectors and Classes you have for one element, the more points you will have, and this is how the information will be displayed.

Flexbox

Flexbox property aims at organizing the elements within your page. It provides an efficient way to lay out, align and distribute space among items in one container. So in order to use this function, you need to organize your HTML code with parent and children elements - because you can apply flex properties to the parents (flex container, how you want the elements in your container to be displayed) and to the children (flex item, how you want the elements to perform).

Properties to the parent:

  • Display:
    You will specify that the display of the elements in your parent container will be flex
    • .container {
    display: flex
}
  • Flex-direction:
    This establishes the main-axis, thus defining the direction flex items are placed in the flex container. Think of flex items as primarily laying out either in horizontal rows or vertical columns.
  • Flex-wrap:
    By default, flex items will all try to fit onto one line. You can change that and allow the items to wrap as needed.
  • Flex-flow:
    Flex-direction + flex-wrap
  • Justify-content:
    This defines the alignment of the elements (aligns elements to the left or right, center, or distribute space between the elements).
  • Align-items:
    This defines the vertical alignment of the elements.
  • Align-content:
    This defines the alignment of container's lines (it only works when you have several lines).

Properties to the children:

  • Order:
    By default, flex items are laid out in the source order. This allows to define the order of the elements.
  • Flex-grow:
    This defines the ability for a flex item to grow if necessary. It dictates what amount of the available space inside the flex container the item should take up.
  • Flex-shrink:
    This defines the ability for a flex item to shrink if necessary.
  • Flex-basis:
    This defines the default size of an element before the remaining space is distributed.
  • Flex:
    Flex-grow + flex-shrink + flex-basis
  • Align-self:
    This allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
